# 알림장 사용 설명서

---

## 실행 방법

터미널(명령 프롬프트)에서 아래 명령어 하나만 입력하세요:

```
uv run main.py
```

---

## 공지사항 입력

실행하면 입력창이 나타납니다. 자유롭게 타이핑하고 **Ctrl+D** 로 완료합니다.

### 기본 조작

| 키 | 동작 |
|---|---|
| `Enter` | 다음 줄로 이동 |
| `위/아래 화살표` | 이전/다음 줄로 이동해서 수정 가능 |
| `좌/우 화살표` | 커서 이동 |
| `Backspace` | 앞 글자 삭제 |
| `Delete` | 뒷 글자 삭제 |
| `Ctrl+Z` | 되돌리기 |
| **`Ctrl+D`** | **입력 완료** |
| `Ctrl+C` | 취소 후 종료 |

---

## 마크다운 단축키

공지사항에 아래 기호를 붙이면 이미지에서 자동으로 꾸며집니다.

### 볼드 (굵게) - `**텍스트**`

양쪽에 별표 두 개씩:

```
**내일 체육복 지참 필수**
```

결과: 하얀 굵은 글씨로 표시

---

### 이탤릭 (기울게) - `*텍스트*`

양쪽에 별표 한 개씩:

```
*수행평가 제출 마감 D-1*
```

결과: 기울어진 회색 글씨로 표시

---

### 취소선 - `~~텍스트~~`

양쪽에 물결표 두 개씩:

```
~~청소 당번: 3모둠 완료~~
```

결과: 회색으로 줄이 그어져 표시

---

### 중요 배지 - `[중요] 텍스트`

앞에 `[중요]` 를 붙임:

```
[중요] 현장학습 동의서 내일까지 제출
```

결과: 빨간 중요 배지 + 흰 굵은 글씨로 강조

---

### 구분선 - `---`

줄 하나를 `---` 만으로 채움:

```
내일 체육복 지참
---
[중요] 동의서 제출
```

결과: 공지 사이에 얇은 구분선 삽입

---

### 입력 예시 (전체)

```
**내일 체육복 지참 필수**
*수행평가 제출 마감 D-1*
---
[중요] 현장학습 동의서 제출
도서관 반납일 확인
~~청소 당번: 3모둠 완료~~
방과후 신청 마감 금요일
```

## 시간표 변동 처리

### 파일 위치

```
alrimjang 폴더 안 > timetable.json
```

### 편집 방법

`timetable.json` 파일을 **메모장**으로 열어서, 맨 아래 닫는 `}` 바로 앞에 아래 내용을 추가합니다.

**변경 전:**
```json
{
    "monday":    ["독서", "생과", "이동D", "이동F", "영독작", "확통"],
    ...
    "friday":    ["영독작", "독서", "이동D", "이동E", "생과", "심국"]
}
```

**변경 후 (예: 5월 22일 금요일 시간표 변동):**
```json
{
    "monday":    ["독서", "생과", "이동D", "이동F", "영독작", "확통"],
    ...
    "friday":    ["영독작", "독서", "이동D", "이동E", "생과", "심국"],

    "overrides": {
        "2026-05-22": ["체육", "수학", "영어", "국어", "과학", "미술"]
    }
}
```

> **주의**
> - friday 줄 끝에 쉼표(,) 추가 필수
> - 날짜 형식: `"2026-05-22"` 처럼 연-월-일 순서
> - 지난 날짜는 지워도 되고 그냥 놔둬도 됩니다

### 여러 날짜 동시 등록 (시험 기간 등)

```json
"overrides": {
    "2026-06-10": ["국어", "수학", "영어", "자습", "자습", "자습"],
    "2026-06-11": ["과학", "사회", "도덕", "자습", "자습"],
    "2026-06-12": ["음악", "미술", "체육", "자습", "자습"]
}
```

## 공휴일 / 학교 자체 휴일 등록

국가 공휴일(설날, 추석, 어린이날 등)은 **자동으로 처리**됩니다. 따로 등록할 필요 없습니다.

개교기념일, 재량휴업일처럼 학교장 권한으로 쉬는 날은 `school_holidays.json`에 등록합니다.

### 파일 위치

```
alrimjang 폴더 안 > school_holidays.json
```

### 편집 방법

메모장으로 열어 날짜와 이름을 추가합니다:

```json
{
    "2026-05-28": "개교기념일",
    "2026-06-01": "재량휴업일",
    "2026-07-24": "여름방학 시작"
}
```

등록하면 해당 날짜를 자동으로 건너뛰고 다음 등교일로 알림장을 작성합니다.

예시: 화요일에 실행 → 수요일이 개교기념일 → 자동으로 목요일 안내사항으로 작성하고 알림 표시
